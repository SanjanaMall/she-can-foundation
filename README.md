# She Can Foundation – Responsive Contact Form Website

A modern and fully responsive contact form webpage developed for the She Can Foundation internship task.

This project focuses on creating a clean, professional, and user-friendly interface while maintaining responsive behavior across desktop, tablet, and mobile devices.

---

## Features

* Fully Responsive Design
* Modern NGO-Inspired UI/UX
* Interactive Contact Form
* Real-Time Character Counter
* Form Validation using JavaScript
* Success Message Animation
* Smooth Scroll-To-Top Button
* Accessible and Clean Layout
* Organized File Structure
* Mobile-Friendly Experience

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts
